Juice Creator

About This Site

Juice Creator is a simple, interactive website made for the Summer of Making programme. It showcases different juice recipes and flavors like strawberry, avocado, and orange in a clean and easy-to-use format.

The purpose of this site is to demonstrate basic web development skills using HTML, CSS, and JavaScript, while keeping the design simple, responsive, and accessible.

Made with ❤️ for the SOM Programme
